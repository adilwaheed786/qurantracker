
# Al-Hafiz: Quran Tracker (الحافظ: قرآن ٹریکر)

**Al-Hafiz** ایک جدید اور خوبصورت ویب ایپ (PWA) ہے جو آپ کو قرآن پاک کے 30 پاروں کی تلاوت کا ریکارڈ رکھنے میں مدد دیتی ہے۔ یہ ایپ خاص طور پر اردو اور عربی صارفین کے لیے **RTL (Right-to-Left)** ڈیزائن کے ساتھ تیار کی گئی ہے۔

## 🌟 اہم خصوصیات (Features)

- **مکمل 30 پارے:** تمام 30 پاروں کی فہرست ان کے اصل عربی ناموں اور اعراب کے ساتھ۔
- **تفصیلی ٹریکنگ:** ہر پارے کو پانچ حصوں میں ٹریک کرنے کی سہولت:
  - شروع (Start)
  - ربع (1/4)
  - نصف (1/2)
  - ثلاثہ (3/4)
  - مکمل (Complete)
- **PWA سپورٹ:** اسے آپ اپنے موبائل پر ایک ایپ کی طرح انسٹال کر سکتے ہیں اور یہ آف لائن بھی کام کرتی ہے۔
- **AI روحانی پیغامات:** گوگل Gemini AI کے ذریعے ہر پارے کے حوالے سے خوبصورت اردو اور انگریزی اقتباسات۔
- **ڈارک موڈ (Dark Mode):** رات کے وقت تلاوت کے ریکارڈ کے لیے بہترین بصری تجربہ۔
- **RTL ڈیزائن:** اردو اور عربی زبان کے مزاج کے مطابق مکمل دائیں سے بائیں ترتیب۔

## 🚀 ٹیکنالوجی (Tech Stack)

- **Frontend:** React.js (v19)
- **Styling:** Tailwind CSS
- **AI Engine:** Google Gemini API (@google/genai)
- **State Management:** React Hooks & LocalStorage
- **Deployment:** PWA (Service Workers & Manifest)

## 🛠️ استعمال کرنے کا طریقہ (How to Use)

1. **ہوم اسکرین:** ایپ کھولتے ہی آپ کو آخری پڑھا ہوا پارہ اور آپ کی مجموعی ترقی نظر آئے گی۔
2. **پارہ منتخب کریں:** کسی بھی پارے پر کلک کریں تاکہ آپ اپنی حالیہ پڑھائی کا نشان لگا سکیں۔
3. **نشان لگائیں:** اپنی پڑھائی کے مطابق (نصف، ربع، وغیرہ) منتخب کر کے "نشان لگائیں" پر کلک کریں۔
4. **خودکار محفوظ:** آپ کی تمام معلومات آپ کے فون میں خودکار طور پر محفوظ رہتی ہیں۔

## 📱 انسٹالیشن (Installation)

یہ ایک **Progressive Web App** ہے۔ اسے انسٹال کرنے کے لیے:
1. اپنے براؤزر (Chrome/Safari) میں لنک کھولیں۔
2. براؤزر مینیو میں جا کر **"Add to Home Screen"** پر کلک کریں۔
3. اب یہ آپ کے موبائل میں ایک ایپ کی طرح ظاہر ہوگی۔
